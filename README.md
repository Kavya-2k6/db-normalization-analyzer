# DB Normalizer 🗃️
**Interactive Database Normalization Tool**

Enter a relational schema and functional dependencies — 
the tool automatically checks 1NF → 2NF → 3NF → BCNF 
with step-by-step explanations and decompositions.

## Features
- Attribute closure computation
- Candidate key & superkey detection  
- Partial dependency detection (2NF)
- Transitive dependency detection (3NF)
- BCNF violation check with decomposition
- Two built-in examples to try instantly

## How to Run
Just open `index.html` in any browser. No installation needed.
